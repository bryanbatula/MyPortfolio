# Software Development Experience: React.js & Node.js

> **Question:** Describe your experience in software development using React.js and Node.js. What applications did you develop and what frameworks did you use?

---

## Overview

My software development experience centers on building full-stack, database-driven web applications using **Node.js** as the server-side runtime and integrating **React.js** on the front end for dynamic, component-based interfaces. Across multiple projects, I have handled everything from requirements gathering and system architecture to deployment and client handoff — giving me a well-rounded, end-to-end perspective on the development lifecycle.

---

## Node.js Experience

Node.js has been my primary back-end runtime across all major projects. I use it together with the **Express.js** framework to build RESTful server logic, handle routing, manage middleware, and connect to databases.

### Key applications built with Node.js + Express:

### 1. Dental Clinic Management System (DCMS)
A full-stack web system independently developed and **sold to a real dental practice**.

- **Back end:** Node.js + Express.js for RESTful API and server-side rendering
- **Templating:** EJS (Embedded JavaScript) for dynamic page generation
- **Database:** PostgreSQL with parameterized queries to prevent SQL Injection
- **Features:** Patient record management, appointment scheduling, treatment history tracking, and an admin dashboard

---

### 2. Barangay Records & Information Management System (BRIMS)
A government-level records system built to solve real administrative bottlenecks in local barangay operations.

- **Back end:** Node.js + Express.js
- **Templating:** EJS
- **Database:** PostgreSQL
- **Features:** Resident record management, document request processing, role-based access control, and digital workflow management
- **Impact:** Replaced manual paper-based processes with a centralized digital system

---

### 3. Secure CRUD Application
A security-focused prototype that demonstrates best practices in back-end development.

- **Back end:** Node.js + Express.js
- **Database:** PostgreSQL
- **Security focus:** Parameterized queries (SQLi prevention), secure session management, and Security-by-Design principles
- **Methodology:** AI-generated code was audited manually using a Human-in-the-Loop approach before integration

---

### 4. Personal Portfolio Website
A server-side rendered personal showcase with custom styling and dynamic templating.

- **Back end:** Node.js + Express.js
- **Templating:** EJS
- **Deployment:** GitHub Pages
- **Live:** [bryanbatula.github.io/MyPortfolio](https://bryanbatula.github.io/MyPortfolio/)

---

### 5. Brew Bites — Café POS System
A live point-of-sale web application for a café with menu management and order processing.

- **Front end:** HTML5, CSS3, JavaScript
- **Deployment:** Vercel
- **Live:** [brew-bites-pos.vercel.app](https://brew-bites-pos.vercel.app/)

## React.js Experience

I have applied React.js in projects requiring highly interactive, component-driven front-end interfaces — particularly in AI-integrated and data-heavy dashboards where a reactive UI is essential.

### Key applications built with or involving React:

### 1. Student Analytics Dashboard *(AI Engineer Portfolio)*
A real-time analytics platform tracking student progress and generating AI-driven insights for instructors.

- **Front end:** React.js for component-based UI and real-time state management
- **Back end:** Node.js + Express REST API
- **Database:** PostgreSQL + AWS
- **Features:** Progress tracking, weak area identification, instructor-facing insight panels

---

### 2. AI Speech Coach *(AI Engineer Portfolio)*
An end-to-end pronunciation analysis system with real-time feedback for English learners.

- **Front end:** React.js for real-time visual feedback components
- **Back end:** Python + FastAPI (integrated with Node.js API layer)
- **AI stack:** Whisper (OpenAI), PyTorch for phoneme alignment
- **Live:** [bryanbatula.github.io/Eng-setup](https://bryanbatula.github.io/Eng-setup/)

---

### 3. Adaptive Grammar Tutor *(AI Engineer Portfolio)*
An LLM-powered grammar correction engine with pedagogical context and adaptive difficulty.

- **Front end:** React.js for interactive lesson interface
- **Back end:** Node.js + LangChain + GPT-4
- **Database:** MongoDB
